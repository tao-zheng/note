## 3.距离度量方法

### 欧式距离

**n维空间点a(x11,x12,⋯,x1n)与b(x21,x22,⋯,x2n)的欧氏距离为：**
$$
d_{12}=\sqrt{\sum_{k = 1}^{n}(x_{1k}-x_{2k})^2}
$$

### 曼哈顿距离

**n维空间点a(x11,x12,⋯,x1n)与b(x21,x22,⋯,x2n)的曼哈顿距离为：**
$$
d_{12}=\sum_{k = 1}^{n}|x_{1k}-x_{2k}|
$$

### 切比雪夫距离

两个点在**各坐标维度上差值的最大值**：对于两个 n 维向量$ \boldsymbol{A} = (a_1, a_2, \dots, a_n) $和 $\boldsymbol{B} = (b_1, b_2, \dots, b_n)$，切比雪夫距离的计算公式为：$d(\boldsymbol{A}, \boldsymbol{B}) = \max\left(|a_1 - b_1|, |a_2 - b_2|, \ldots, |a_n - b_n|\right)$



## 4.特征预处理

### 为什么进行归一化、标准化

特征的**单位或者大小相差较大，或者某特征的方差相比其他的特征要大出几个数量级**，**容易影响（支配）目标结果**，使得一些模型（算法）无法学习到其它的特征。

### 归一化

通过对原始数据进行变换把数据映射到【mi,mx】(默认为[0,1])之间
$$
X' = \frac{x - \min}{\max - \min}
$$
**归一化受到最大值与最小值的影响，这种方法容易受到异常数据的影响, 鲁棒性较差，适合传统精确小数据场景**

### 标准化

通过对原始数据进行标准化，转换为均值为0标准差为1的标准正态分布的数据
$$
X' = \frac{x - \text{mean}}{\sigma}
$$
**对于标准化来说，如果出现异常点，由于具有一定数据量，少量的异常点对于平均值的影响并不大**

## 5.利用KNN算法进行鸢尾花分类

鸢尾花数据集包含了150条鸢尾花信息，每50条取自三个鸢尾花中之一：**Versicolour、Setosa和Virginica**

每个花的特征用如下属性描述：**Sepal length 、 Sepal width 、 Petal length 、 Petal width**

```python

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

# 1.加载数据集
iris_data = load_iris()
# print(iris_data)

x_train, x_test, y_train, y_test = train_test_split(iris_data.data, iris_data.target, test_size=0.3, random_state=22)# 随机种子
```

## 6.超参数选择的方法

### 交叉验证（得到更加准确可信的模型评分）

交叉验证是一种数据集的分割方法，将训练集划分为 n 份，其中一份做验证集、其他n-1份做训练集

![image-20230831163236810](D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day02-KNN%E7%AE%97%E6%B3%95/03-%E7%AC%94%E8%AE%B0/images/image-20230831163236810.png)

总共训练并评估n次后，**取测试结果平均值做为模型得分**

### 网格搜索（寻找最优超参数）

只需要将若干参数传递给网格搜索对象，它自动帮我们完成不同超参数的组合、模型训练、模型评估，最终返回一组最优的超参数。

**交叉验证网格搜索在鸢尾花分类中的应用**：

```python
# 0.导入工具包
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split,GridSearchCV
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# 1.加载数据
data = load_iris()

# 2 数据集划分
x_train,x_test,y_train,y_test=train_test_split(data.data,data.target,test_size=0.2,random_state=22)

# 3.特征预处理
pre = StandardScaler()
x_train=pre.fit_transform(x_train)
x_test=pre.transform(x_test)

# 4.模型实例化+交叉验证+网格搜索
model = KNeighborsClassifier(n_neighbors=1)
paras_grid = {'n_neighbors':[4,5,7,9]}

estimator =GridSearchCV(estimator=model,param_grid=paras_grid,cv=4)

model = estimator.best_estimator_

print(estimator.best_estimator_) #返回使用最优参数训练的模型实例。
print(estimator.best_params_：) # 返回最优参数组合的字典。


y_prdict=model.predict(x_test)

print(accuracy_score(y_test,y_prdict))

```

## 1.线性回归介绍

### 线性回归

线性回归是利用 **回归方程对 一个或多个自变量(特征值)和因变量(目标值)之间** 关系进行建模的一种分析方式。

注意事项：

1 **为什么叫线性模型？**因为求解的w，都是w的零次幂（常数项）所以叫成线性模型

线性回归模型虽然被称为 “线性”，但它的 “线性” 指的是**对参数（如 a、b、c）的线性组合**，而非对特征（如 x）的线性关系。因此，线性回归**完全可以拟合二次函数 \(ax^2 + bx + c\)**，只需通过特征工程将非线性特征转换为线性形式。只需令x1=x，x2=x^2，模型变为：y=c+bx1+ax2；

2 在线性回归中，从数据中获取的规律其实就是学习权重系数w

3 某一个权重值w越大，说明这个权重的数据对房子价格影响越大 

### 线性回归分类

## 2.线性回归问题的求解

### 梯度下降算法

#### 梯度下降算法思想

什么是梯度下降法

• 求解函数极值还有更通用的方法就是梯度下降法。顾名思义:沿着梯度下降的方向求解极小值 

![image-20230901183007785](D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day03-%E7%BA%BF%E6%80%A7%E5%9B%9E%E5%BD%92/03-%E7%AC%94%E8%AE%B0/images/image-20230901183007785.png)

 最终找到最优解 这个方法可用来求损失函数最优解， 比正规方程更通用

### 梯度下降公式
- 循环迭代求当前点的梯度，更新当前的权重参数
  $$
  \theta_{i + 1}=\theta_{i}-\alpha\frac{\partial}{\partial\theta_{i}}J(\theta)
  $$
  a：学习率（步长）不能太大，也不能太小。机器学习中：0.001-0.01，深度学习中:10e-6

- 梯度是上升最快的方向，我们需要是下降最快的方向，所以需要加负号

### 正则化

#### **L1正则化**

- 假设MSE(w)是未加正则项的损失，α是一个超参，控制正则化项的大小。

- 则最终的损失函数：
  $$
  J(w) = \text{MSE}(w) + \alpha \sum_{i = 1}^{n} |w_i|
  $$





作用：用来进行特征选择，主要原因在于L1正则化会使得较多的参数为0，从而产生稀疏解,可以将0对应的特征遗弃，进而用来选择特征。

LASSO回归:

```python
# 在sklearn.linear_model.LinearRegression的基础上，添加了L1正则化

```

#### **L2正则化**

- 假设MSE(w)是未加正则项的损失，α是一个超参，控制正则化项的大小。

- 则最终的损失函数：
  $$
  J(w) = \text{MSE}(w) + \alpha\sum_{i = 1}^{n} w_i^2
  $$





作用：L2正则化会使得权重趋向于0，一般不等于0

优点：越小的参数说明模型越简单，越简单的模型则越不容易产生过拟合现象

### 极大似然估计

核心思想：设模型中含有待估参数w，可以取很多值。已经知道了样本观测值，从w的一切可能值中（选出一个使该观察值出现的概率为最大的值，作为w参数的估计值，这就是极大似然估计。举个例子：

假设有一枚不均匀的硬币，出现正面的概率和反面的概率是不同的。假定出现正面的概率为𝜃， 抛了6次得到如下现象 D = {正面，反面，反面，正面，正面，正面}。每次投掷事件都是相互独立的。 则根据产生的现象D，来估计参数𝜃是多少?

```
P(D|𝜃) = P {正面，反面，反面，正面，正面，正面}
 = P(正面|𝜃) P(正面|𝜃) P(正面|𝜃) P(正面|𝜃) P(正面|𝜃) P(正面|𝜃)

=𝜃 *(1-𝜃)*(1-𝜃)𝜃*𝜃*𝜃 = 𝜃4(1 − 𝜃)
```

问题转化为:求此函数的极大值时，估计𝜃为多少

## 2.逻辑回归原理

### 原理

逻辑回归概念 Logistic Regression

• **一种分类模型，把线性回归的输出，作为逻辑回归的输入。** 

• 输出是(0, 1)之间的值

• 基本思想

1. 利用线性模型 f(x) = wx + b 根据特征的重要性计算出一个值
2. 再使用 sigmoid /softmax函数将 f(x) 的输出值映射为概率值
   1. 设置阈值(eg:0.5)，输出概率值大于 0.5，则将未知样本输出为 1 类
   2. 否则输出为 0 类

   3.逻辑回归的假设函数
​    h(w) = sigmoid(wx + b )

**在逻辑回归中，当预测结果不对的时候，我们该怎么衡量其损失呢？**

### 损失函数：交叉熵

#### **数学定义**

对数损失函数（也称为交叉熵损失）的表达式为：$L(\mathbf{w}, b) = -\frac{1}{n} \sum_{i=1}^n \left[ y_i \log(\hat{p}_i) + (1-y_i) \log(1-\hat{p}_i) \right]$ 。

#### **直观解释**

对数损失的本质是**极大似然估计**的负数。通过最大化样本的似然函数，等价于最小化对数损失。实际中通常最大化**对数似然函数**：$\log L(\mathbf{w}, b) = \sum_{i=1}^n \left[ y_i \log \hat{p}_i + (1-y_i) \log (1-\hat{p}_i) \right]$，而逻辑回归的损失函数是**负对数似然**（因为优化目标通常是 “最小化损失”）：$\text{Loss} = -\log L(\mathbf{w}, b)$，因此，“最小化对数损失” 与 “最大化似然函数” 是完全等价的。

True Positive ：表示样本真实的类别

### Precision（精确率）

精确率也叫做查准率，指的是对正例样本的预测准确率。比如：我们把恶性肿瘤当做正例样本，则我们就需要知道模型对恶性肿瘤的预测准确率。
$$
P = \frac{TP}{TP + FP}
$$

### Recall（召回率）

召回率也叫做查全率，指的是预测为真正例样本占所有真实正例样本的比重。例如：我们把恶性肿瘤当做正例样本，则我们想知道模型是否能把所有的恶性肿瘤患者都预测出来。
$$
P = \frac{TP}{TP + FN}
$$

### F1-score

如果我们对模型的精度、召回率都有要求，希望知道模型在这两个评估方向的综合预测能力如何？则可以使用 F1-score 指标。
$$
F1 = \frac{2TP}{2TP + FN + FP}=\frac{2\cdot\text{Precision}\cdot\text{Recall}}{\text{Precision}+\text{Recall}}
$$

### ROC曲线和AUC指标

ROC 曲线：我们分别考虑正负样本的情况：

1. 正样本中被预测为正样本的概率，即：TPR （True Positive Rate）
2. 负样本中被预测为正样本的概率，即：FPR （False Positive Rate）

# 决策树

## 1.决策树简介

**决策树是什么？**

决策树是一种树形结构，树中每个内部节点表示一个特征上的判断，每个叶子节点代表一种分类结果

**决策树的建立过程**：

1.特征选择：选取有较强分类能力的特征。

2.决策树生成：根据选择的特征生成决策树。

3.决策树也易过拟合，采用剪枝的方法缓解过拟合。

## 2.ID3决策树

### 信息熵

**ID3 树是基于信息增益构建的决策树.**

信息熵公式：
$$
\large
Entropy(S) = -\sum_{i=1}^{k}p_i\log(p_i)
$$
例子1：假如有三个类别，分别占比为：{1/3,1/3,1/3}，信息熵计算结果为：
$$
S = - \sum_{i = 0}^{n} P(x_i) \log_2 P(x_i) = \left(- \frac{1}{3} \log_2 \frac{1}{3}\right) * 3 = 1.0986
$$

### **条件熵**

条件熵表示在已知某个特征 A 的取值后，数据集 S 的熵。假设特征 A 有 v 个可能取值，将 S 划分为 v 个子集$ S_1, S_2, \dots, S_v$，则条件熵为：
$$
\text{Entropy}(S|A) = \sum_{i=1}^v \frac{|S_i|}{|S|} \cdot \text{Entropy}(S_i)\
$$
其中 $|S_i|$是子集的$S_i$样本数，$S$是总样本数。

### 信息增益

信息增益定义为划分前的熵减去划分后的条件熵，即：
$$
\text{Gain}(S, A) = \text{Entropy}(S) - \text{Entropy}(S|A)
$$
直观理解：信息增益越大，说明使用特征 A 划分后，数据的纯度提升得越多。

根据信息增益选择特征方法是：**对训练数据集D，计算其每个特征的信息增益，并比较它们的大小，并选择信息增益最大的特征进行划分**。**表示由于特征A而使得对数据D的分类不确定性减少的程度。**

### ID3树构建流程

假设有以下数据集，根据天气特征预测是否去打球：

| 天气（Outlook） | 温度（Temp） | 湿度（Humidity） | 风速（Wind） | 是否打球（Play） |
| --------------- | ------------ | ---------------- | ------------ | ---------------- |
| Sunny           | Hot          | High             | Weak         | No               |
| Sunny           | Hot          | High             | Strong       | No               |
| Overcast        | Hot          | High             | Weak         | Yes              |
| Rain            | Mild         | High             | Weak         | Yes              |
| Rain            | Cool         | Normal           | Weak         | Yes              |
| Rain            | Cool         | Normal           | Strong       | No               |
| Overcast        | Cool         | Normal           | Strong       | Yes              |

#### **计算根节点的熵**

总样本数 \(|S|=7\)，其中 4 个 Yes，3 个 No，熵为：$\text{Entropy}(S) = -\frac{4}{7}\log_2\frac{4}{7} - \frac{3}{7}\log_2\frac{3}{7} \approx 0.985$

#### 计算各特征的信息增益

以 “天气（Outlook）” 为例，其取值为 Sunny、Overcast、Rain，分别对应 2、2、3 个样本：

- Sunny：2 个 No，熵为 0；
- Overcast：2 个 Yes，熵为 0；
- Rain：2 个 Yes，1 个 No，熵为$ -\frac{2}{3}\log_2\frac{2}{3} - \frac{1}{3}\log_2\frac{1}{3} \approx 0.918$。

条件熵为：$\text{Entropy}(S|\text{Outlook}) = \frac{2}{7} \cdot 0 + \frac{2}{7} \cdot 0 + \frac{3}{7} \cdot 0.918 \approx 0.393$

信息增益为：$\text{Gain}(S, \text{Outlook}) = 0.985 - 0.393 = 0.592$

同理计算其他特征的信息增益，发现 “Outlook” 的增益最大（0.592），因此选择它作为根节点的划分特征。



## 3.C4.5决策树

**ID3树的不足：偏向于选择种类多的特征作为分裂依据** 

### 信息增益率

**C4.5 引入信息增益率**，通过惩罚取值多的特征来解决这个问题：$\text{GainRatio}(S, A) = \frac{\text{Gain}(S, A)}{\text{SplitInfo}(S, A)}$其中：

- $\text{Gain}(S, A)$是信息增益（同 ID3）；
- $\text{SplitInfo}(S, A) = -\sum_{i=1}^v \frac{|S_i|}{|S|} \log_2 \frac{|S_i|}{|S|}$是分裂信息，衡量特征 A 自身的熵。

### 支持连续数值特征

C4.5 通过**二分法**处理连续特征：对连续特征 A 的所有取值排序，计算相邻两个值的中间点作为候选分裂点（如$ a_i$和$ a_{i+1}$的中点$ \frac{a_i + a_{i+1}}{2}$，选择信息增益率最大的分裂点作为划分阈值。

### 剪枝优化

C4.5 引入**后剪枝**减少过拟合：

### 算法流程

1. **特征选择**：
   - 对离散特征，计算信息增益率$ \text{GainRatio}(S, A)$；
   - 对连续特征，先排序并找到最优分裂点，再计算信息增益率；
   - 选择增益率最大的特征（或分裂点）作为划分依据。
2. **数据集划分**：
   - 对离散特征，按每个取值划分数据集；
   - 对连续特征，按 “≤阈值” 和 “> 阈值” 二分数据集。
3. **递归生成树**：
   - 对每个子节点，递归执行特征选择和划分，直到满足终止条件（如样本属于同一类，或无剩余特征）。
4. **后剪枝优化**：
   - 使用独立验证集评估每个内部节点的剪枝必要性，替换为叶节点以提升泛化能力。

### 天气分类

假设有以下数据集（新增温度为连续值）：

| 天气（Outlook） | 温度（Temp，连续） | 湿度（Humidity） | 风速（Wind） | 是否打球（Play） |
| --------------- | ------------------ | ---------------- | ------------ | ---------------- |
| Sunny           | 85                 | High             | Weak         | No               |
| Sunny           | 80                 | High             | Strong       | No               |
| Overcast        | 83                 | High             | Weak         | Yes              |
| Rain            | 70                 | High             | Weak         | Yes              |
| Rain            | 68                 | Normal           | Weak         | Yes              |
| Rain            | 65                 | Normal           | Strong       | No               |

#### 连续特征处理

- 排序温度值：\(65, 68, 70, 80, 83, 85\)
- 候选分裂点：\(66.5, 69, 75, 81.5, 84\)
- 计算每个分裂点的信息增益率（过程略），假设最优分裂点为 75，则将温度划分为 “≤75” 和 “>75”。

#### 特征选择

假设计算所有特征的信息增益率后，“天气（Outlook）” 的增益率最高，作为根节点划分特征。

#### **生成决策树**

递归划分后，可能生成如下决策树：

```plaintext
[Outlook = Sunny] → [Humidity = High] → No  
                 → [Humidity = Normal] → Yes  
[Outlook = Overcast] → Yes  
[Outlook = Rain] → [Temp ≤ 75] → Yes  
               → [Temp > 75] → No
```



## 5.CART决策树

### CART 的核心特点

与 ID3、C4.5 相比，CART 有三个显著差异：

1. **树结构**：始终生成**二叉树**（每个内部节点只有两个子节点，即 “是 / 否” 判断）。
2. 分裂准则：
   - 分类任务：使用**基尼不纯度** 而非信息熵；
   - 回归任务：使用**均方误差** 。
3. **剪枝策略**：采用**代价复杂度剪枝**，通过数学公式量化剪枝代价，更客观。

### 分类树：基尼不纯度

**基尼不纯度的定义：基尼不纯度表示从数据集中随机抽取两个样本，其类别不同的概率**。对于包含 K 个类别的数据集 S，基尼不纯度为：$\text{Gini}(S) = 1 - \sum_{k=1}^K p_k^2$，其中$ p_k$是第 k 类样本在 S 中的占比。

**极端情况**：

- 若所有样本属于同一类（纯节点），则 $\text{Gini}(S) = 0$；
- 若两类样本各占一半（二分类），则$\text{Gini}(S) = 1 - (0.5^2 + 0.5^2) = 0.5$（最大混乱）。

CART 通过**最小化子节点的加权基尼不纯度**选择最优分裂特征和阈值：

1. 对特征 A 的某个可能分裂（如离散特征的某个取值，或连续特征的某个阈值），将数据集 S 划分为左子树 $S_L$和右子树$S_R$；
2. 计算分裂后的加权基尼不纯度：$\text{Gini}(S, A) = \frac{|S_L|}{|S|} \cdot \text{Gini}(S_L) + \frac{|S_R|}{|S|} \cdot \text{Gini}(S_R)$
3. 选择使$\text{Gini}(S, A)$最小的分裂方式。

### 回归树：均方误差

对于回归任务，CART 使用**均方误差**衡量分裂效果：

1. 对特征 A 的某个分裂，将数据集划分为 $S_L$和$S_R$；
2. 计算分裂后的加权 MSE：$\text{MSE}(S, A) = \frac{|S_L|}{|S|} \cdot \text{MSE}(S_L) + \frac{|S_R|}{|S|} \cdot \text{MSE}(S_R)$其中$ \text{MSE}(S_L) = \frac{1}{|S_L|} \sum_{x \in S_L} (y - \bar{y}_L)^2$，$\bar{y}_L$是$S_L$中目标值的均值；
3. 选择使$\text{MSE}(S, A)$最小的分裂方式。

### CART 树的生成流程

CART 采用**递归二分法**生成二叉树，流程如下：

1. **初始化**：以数据集 S 为根节点。
2. 终止条件判断：
   - 若所有样本的特征或目标值相同，停止分裂，标记为叶节点（分类树取多数类，回归树取均值）；
   - 若达到预设树深度或样本数阈值，停止分裂。
3. 最优分裂选择：
   - 对每个特征 A，遍历所有可能的分裂点（离散特征的每个取值，连续特征的所有相邻值中点）；
   - 计算每个分裂点的基尼不纯度（分类）或 MSE（回归）；
   - 选择最优分裂点，将数据集划分为左子树 \(S_L\) 和右子树 \(S_R\)。
4. 递归生成子树：
   - 对 $S_L$和$S_R$分别递归执行步骤 2-3，生成左右子节点。

### 示例

数据集如下（共 8 个样本）：

| 序号 | \(x_1\)（年龄） | \(x_2\)（有电脑） | \(x_3\)（零花钱） | y（购买） |
| ---- | --------------- | ----------------- | ----------------- | --------- |
| 1    | 15              | 1                 | 500               | 1         |
| 2    | 18              | 1                 | 800               | 1         |
| 3    | 25              | 0                 | 1000              | 0         |
| 4    | 30              | 0                 | 1200              | 0         |
| 5    | 12              | 1                 | 300               | 1         |
| 6    | 22              | 1                 | 600               | 0         |
| 7    | 16              | 0                 | 700               | 0         |
| 8    | 19              | 1                 | 900               | 1         |

**CART 树构建步骤**

CART 树是**二叉树**，通过**基尼不纯度**选择最优分裂，每次将数据分为左右两部分。

**步骤 1：计算根节点的基尼不纯度**

根节点包含所有 8 个样本，其中 \(y=1\) 的有 4 个（买），\(y=0\) 的有 4 个（不买）：$\text{Gini}(根节点) = 1 - \left( \left(\frac{4}{8}\right)^2 + \left(\frac{4}{8}\right)^2 \right) = 1 - (0.25 + 0.25) = 0.5$

**步骤 2：选择最优分裂特征和阈值**

需遍历所有特征的所有可能分裂点，计算分裂后的**加权基尼不纯度**，选择最小值对应的分裂。

**特征$x_1$（年龄，连续值）**

对年龄排序：12,15,16,18,19,22,25,30，候选阈值为相邻值的中点（13.5,15.5,17,18.5,20.5,23.5,27.5）。

以阈值 18.5（年龄≤18.5 vs >18.5）为例：

- 左子树：年龄≤18.5（样本 1、2、5、7，共 4 个，\(y=1\) 有 3 个，\(y=0\) 有 1 个）
- 右子树：年龄 > 18.5（样本 3、4、6、8，共 4 个，\(y=1\) 有 1 个，\(y=0\) 有 3 个）

加权基尼不纯度：$\text{Gini}(x_1=18.5) = \frac{4}{8} \cdot \left[1 - \left(\frac{3}{4}\right)^2 - \left(\frac{1}{4}\right)^2\right] + \frac{4}{8} \cdot \left[1 - \left(\frac{1}{4}\right)^2 - \left(\frac{3}{4}\right)^2\right]= \frac{1}{2} \cdot 0.375 + \frac{1}{2} \cdot 0.375 = 0.375$

**特征$x_2$（是否有电脑，离散值）**

$x_2$有两个取值（0 和 1），分裂方式为：

- 左子树：\(x_2=0\)（样本 3、4、7，共 3 个，其中 \(y=0\) 有 3 个，\(y=1\) 有 0 个）
- 右子树：\(x_2=1\)（样本 1、2、5、6、8，共 5 个，其中 \(y=1\) 有 4 个，\(y=0\) 有 1 个）

计算加权基尼不纯度：$(\text{Gini}(x_2) = \frac{3}{8} \cdot \left[1 - (1^2 + 0^2)\right] + \frac{5}{8} \cdot \left[1 - \left(\frac{4}{5}\right)^2 - \left(\frac{1}{5}\right)^2\right]= \frac{3}{8} \cdot 0 + \frac{5}{8} \cdot \left(1 - 0.64 - 0.04\right) = 0 + \frac{5}{8} \cdot 0.32 = 0.2$

**特征$x_3$（零花钱，连续值）**

计算后发现其最优分裂的加权基尼不纯度为 0.3（过程略）。

**最优分裂选择**

对比三个特征的最优分裂结果：$x_2$最小，因此，**根节点选择 $x_2$作为分裂特征**，分裂规则为：

- 左子树：\(x_2=0\)（无电脑）
- 右子树：\(x_2=1\)（有电脑）

**步骤 3：递归分裂子节点**

**左子树（\(x_2=0\)，无电脑）**

包含样本 3、4、7（均为 \(y=0\)，基尼不纯度 = 0），已是**纯节点**，无需继续分裂，标记为叶节点（预测值 = 0）。

**右子树（\(x_2=1\)，有电脑）**

包含样本 1、2、5、6、8（\(y=1\) 有 4 个，\(y=0\) 有 1 个），基尼不纯度 = 0.32（非纯节点），需继续分裂。

对该子树计算各特征的最优分裂，发现**年龄 \(x_1=20\)** 是最佳分裂点：

- 左子树：年龄≤20（样本 1、2、5、8，均为 \(y=1\)，基尼 = 0）
- 右子树：年龄 > 20（样本 6，\(y=0\)，基尼 = 0）

均为纯节点，停止分裂。

### **最终 CART 树结构**

```plaintext
根节点：x2（是否有电脑）
├─ x2=0（无电脑）→ 叶节点：预测=0（不买）
│  （样本3、4、7，基尼=0）
│
└─ x2=1（有电脑）→ 内部节点：x1（年龄）≤20？
   ├─ 是 → 叶节点：预测=1（买）
   │  （样本1、2、5、8，基尼=0）
   │
   └─ 否 → 叶节点：预测=0（不买）
      （样本6，基尼=0）
```

## 7.决策树剪枝

### 什么是剪枝?

剪枝是决策树学习算法对付 **过拟合** 的主要手段。

### 常见减枝方法汇总

决策树剪枝的基本策略有"预剪枝" (pre-pruning）和"后剪枝"（post- pruning) 。

1. 预剪枝是指在决策树生成过程中，对每个结点在划分前先进行估计，若当前结点的划分不能带来决策树泛化性能提升，则停止划分并将当前结点标记为叶结点;
2. 后剪枝则是先从训练集生成一棵完整的决策树，然后自底向上地对非叶结点进行考察，若将该结点对应的子树替换为叶结点能带来决策树泛化性能提升，则将该子树替换为叶结点。

### 剪枝方法对比

**预剪枝优点：**

- 预剪枝使决策树的很多分支没有展开，不单降低了过拟合风险，还显著减少了决策树的训练、测试时间开销

**后剪枝优点：**

- 比预剪枝保留了更多的分支。一般情况下，后剪枝决策树的欠拟合风险很小，泛化性能往往优于预剪枝

## 2.Adaboost

### Adaboost算法简介

Adaptive Boosting基于 Boosting思想实现的一种集成学习算法，核心思想是通过逐步提高那些被前一步分类错误的样本的权重来训练一个强分类器。其特点是：**训练时，样本具有权重，并且在训练过程中动态调整。被分错的样本会加大权重，算法更加关注难分的样本。**

**模型权重更新公式:**
$$
\alpha_t = \frac{1}{2} \ln\left( \frac{1 - \varepsilon_t}{\varepsilon_t} \right)
$$

ε<sub>t</sub> 表示第 t 个弱学习器的错误率

**样本权重更新公式:**
$$
D_{t+1}(x) = \frac{D_t(x)}{Z_t} * 
\begin{cases} 
e^{-\alpha_t}, & \text{预测值 = 真实值} \\
e^{\alpha_t}, & \text{预测值} \neq \text{真实值}
\end{cases}
$$

1. Z<sub>t</sub> 为归一化值（所有样本权重的总和）
2. D<sub>t</sub>(x) 为样本权重
3. α<sub>t</sub> 为模型权重。

**最终的强分类器:**
$$
H(x) = \text{sign}\left( \sum_{m=1}^M \alpha_m \cdot h_m(x) \right)
$$

## 3.GBDT 

### 梯度提升树

![1752628450763](C:\Users\17630\AppData\Roaming\Typora\typora-user-images\1752628450763.png)

### GBDT例子

- **任务**：预测房屋价格（万元），特征为房屋面积（平方米）。

- 训练数据：

  | 样本序号 | 面积 x（㎡） | 真实房价 y（万元） |
  | -------- | ------------ | ------------------ |
  | 1        | 50           | 50                 |
  | 2        | 80           | 80                 |
  | 3        | 120          | 120                |
  | 4        | 150          | 150                |

- **弱学习器**：使用最简单的决策树桩（深度为 1 的决策树，只有 1 个分裂节点）。

- **损失函数**：平方损失$L(y, \hat{y}) = \frac{1}{2}(y - \hat{y})^2$（此时残差 = 真实值 - 预测值）。

- **迭代轮数**：3 轮（生成 3 棵树）。

- **学习率**：$\eta$ = 0.5（控制每棵树的修正幅度，避免过拟合）。

**第 0 轮：初始化模型（初始预测）**

GBDT 从一个简单的初始模型开始，通常是所有样本的均值：$F_0(x) = \frac{50 + 80 + 120 + 150}{4} = 100 $

残差计算：$r_{0,i} = y_i - F_0(x_i) = [50-100, 80-100, 120-100, 150-100] = [-50, -20, +20, +50]$

**第 1 轮：训练第 1 棵树 $h_1(x)$（拟合第 0 轮残差**）

目标：找到最优决策树桩 $h_1(x)$，尽可能拟合残差 $r_{0,i}$。

1. **寻找最优分裂点**： 尝试按面积 x 分裂，计算不同分裂点的误差（平方损失）：

   - 若分裂点为(x=100\)：
     - 左子树（$x \leq 100$）：样本 1、2，残差均值 $\frac{-50 + (-20)}{2} = -35$
     - 右子树（$x > 100$）：样本 3、4，残差均值 $\frac{20 + 50}{2} = 35$
     - 误差：$(-50 +35)^2 + (-20 +35)^2 + (20 -35)^2 + (50 -35)^2 = 0$（完美拟合残差）。

   选择分裂点 \(x=100\)，得到第 1 棵树：$h_1(x) = \begin{cases}     -35 & \text{若 } x \leq 100 \\    +35 & \text{若 } x > 100     \end{cases}$

2. **更新模型**（学习率 $\eta=0.5$）：$F_1(x) = F_0(x) + \eta \cdot h_1(x)$

   - 样本 1（50㎡）：$F_1 = 100 + 0.5 \times (-35) = 82.5$
   - 样本 2（80㎡）：$F_1 = 100 + 0.5 \times (-35) = 82.5$
   - 样本 3（120㎡）：$F_1 = 100 + 0.5 \times 35 = 117.5$
   - 样本 4（150㎡）：$F_1 = 100 + 0.5 \times 35 = 117.5$

3. **计算第 1 轮残差**（用于下一轮训练）：$r_{1,i} = y_i - F_1(x_i) = [50-82.5, 80-82.5, 120-117.5, 150-117.5] = [-32.5, -2.5, +2.5, +32.5]$

**第 2 轮：训练第 2 棵树 $h_2(x)$（拟合第 1 轮残差）**

目标：用新树桩拟合残差 $r_{1,i} = [-32.5, -2.5, +2.5, +32.5]$。

1. **寻找最优分裂点**： 尝试分裂点 \(x=90\)（将样本 1、2 分开）：
   - 左子树（$x \leq 90$）：样本 1，残差均值 \(-32.5\)
   - 右子树（$x > 90$）：样本 2、3、4，残差均值 $\frac{-2.5 + 2.5 + 32.5}{3} \approx 10.83$误差最小，得到第 2 棵树：$h_2(x) = \begin{cases}     -32.5 & \text{若 } x \leq 90 \\    +10.83 & \text{若 } x > 90     \end{cases}$
2. **更新模型**：$F_2(x) = F_1(x) + \eta \cdot h_2(x)$
   - 样本 1（50㎡）：$F_2 = 82.5 + 0.5 \times (-32.5) = 66.25$
   - 样本 2（80㎡）：$F_2 = 82.5 + 0.5 \times 10.83 \approx 87.92$
   - 样本 3（120㎡）：$F_2 = 117.5 + 0.5 \times 10.83 \approx 122.92$
   - 样本 4（150㎡）：$F_2 = 117.5 + 0.5 \times 10.83 \approx 122.92$
3. **计算第 2 轮残差**：$r_{2,i} = y_i - F_2(x_i) \approx [50-66.25, 80-87.92, 120-122.92, 150-122.92] \approx [-16.25, -7.92, -2.92, +27.08]$

**第 3 轮：训练第 3 棵树 $h_3(x)$（拟合第 2 轮残差）**

目标：拟合残差 $r_{2,i} \approx [-16.25, -7.92, -2.92, +27.08]$。

1. **最优分裂点**：选择 \(x=130\)（将样本 4 单独划分）：

   - 左子树（$x \leq 130$）：样本 1、2、3，残差均值 $\frac{-16.25 -7.92 -2.92}{3} \approx -9.03$
   - 右子树（$x > 130$）：样本 4，残差均值 \(+27.08\)

   第 3 棵树：$h_3(x) = \begin{cases}     -9.03 & \text{若 } x \leq 130 \\    +27.08 & \text{若 } x > 130     \end{cases}$

2. **更新模型**：$F_3(x) = F_2(x) + \eta \cdot h_3(x)$

   - 样本 1：$66.25 + 0.5 \times (-9.03) \approx 61.74$
   - 样本 2：$87.92 + 0.5 \times (-9.03) \approx 83.41$
   - 样本 3：$122.92 + 0.5 \times (-9.03) \approx 118.41$
   - 样本 4：$122.92 + 0.5 \times 27.08 \approx 136.46$

## 4.XGBoost

XGBoost（Extreme Gradient Boosting）全名叫极端梯度提升树，XGBoost是集成学习方法的王牌。

### XGBoost算法思想

XGBoost 是对GBDT的改进：

1. 求解损失函数极值时使用泰勒二阶展开
2. 在损失函数中加入了正则化项，提高模型对未知数据的泛化能力。

### XGboost的目标函数

XGBoost（Extreme Gradient Boosting）是对梯度提升树的改进，并且在损失函数中加入了正则化项。
$$
\mathrm{obj}(\theta)=\sum_{i}^{n}L(y_i,\hat{y}_i)+\sum_{k = 1}^{K}\Omega(f_k)
$$
目标函数的第一项表示整个强学习器的损失，第二部分表示强学习器中 K 个弱学习器的复杂度。

xgboost 每一个弱学习器的复杂度主要从两个方面来考量：
$$
\Omega(f)=\gamma T+\frac{1}{2}\lambda \| w \|^2
$$

1. γT 中的 T 表示一棵树的叶子结点数量
2. λ||w||<sup>2</sup> 中的 w 表示叶子结点权重

### XGboost API

```python
bst = XGBClassifier(n_estimators, max_depth, learning_rate, objective)
```

# 朴素贝叶斯

## 1.朴素贝叶斯介绍

**朴素贝叶斯是基于贝叶斯定理和特征条件独立假设的分类算法**。它假设特征之间相互独立，这一假设在实际应用中往往不成立，但朴素贝叶斯算法在文本分类、垃圾邮件过滤、情感分析等领域表现出色，因其简单高效且需要较少的训练数据。

### 核心原理

#### 贝叶斯定理

贝叶斯定理是朴素贝叶斯算法的基础，公式如下：
$$
P(Y|X) = \frac{P(X|Y) \cdot P(Y)}{P(X)}
$$
其中：

- `P(Y|X)` 是后验概率，即给定特征 X 的情况下类别 Y 的概率。
- `P(X|Y)` 是似然度，即给定类别 Y 的情况下特征 X 的概率。
- `P(Y)` 是先验概率，即类别 Y 的初始概率。
- `P(X)` 是证据，用于归一化结果。

#### 朴素假设

朴素贝叶斯的 "朴素" 体现在它假设特征之间相互独立，即：
$$
P(X|Y) = P(X_1|Y) \cdot P(X_2|Y) \cdot \ldots \cdot P(X_n|Y)
$$

#### 分类规则

对于新样本，计算其属于每个类别的后验概率，选择概率最大的类别作为预测结果：
$$
\hat{Y} = \arg\max_Y P(Y) \cdot \prod_{i=1}^n P(X_i|Y)
$$

### 拉普拉斯平滑系数

**避免零概率**：以多项式朴素贝叶斯在文本分类中的应用为例，设训练数据集中有C个类别，对于某一个类别y ，其训练样本数量为Ny，特征词表大小为V，某个特征词xi在类别y的训练样本中出现的次数为count(xi,y) 。使用拉普拉斯平滑后，引入平滑系数α（通常α取值为 1），条件概率变为
$$
P(X_i|Y) = \frac{count(X_i, Y) + \alpha}{N_Y + \alpha \cdot V}
$$

# k-means聚类

## 1.聚类算法简介

### 聚类算法介绍

一种典型的无监督学习算法，主要用于将相似的样本自动归到一个类别中。

### 算法步骤

1. **初始化簇中心**：随机选择K个数据点作为初始的簇中心。
2. **分配数据点**：遍历数据集中的每个数据点，计算它与各个簇中心的距离，将其分配到距离最近的簇中。
3. **更新簇中心**：根据每个簇内的数据点，重新计算各个簇的中心位置（通常是取簇内所有数据点的均值）。
4. **判断收敛**：检查簇中心是否发生变化（如变化小于某个阈值，或达到预设的最大迭代次数）。如果满足收敛条件，则算法结束；否则，返回步骤 2 继续迭代。

## 2.聚类API的初步使用

```python

# 参数:
  # n_clusters:开始的聚类中心数量
```

## 3.评价指标

###  SSE-误差平方和

K 表示聚类中心的个数，C<sub>i</sub> 表示簇，p 表示样本，m<sub>i</sub> 表示簇的质心

<img src="D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day09-%E8%81%9A%E7%B1%BBkmeans%E7%AE%97%E6%B3%95+SVM/03-%E7%AC%94%E8%AE%B0/images/image-20230907110135000.png" alt="image-20230907110135000" style="zoom:67%;" />

SSE 越小，表示数据点越接近它们的中心，聚类效果越好。

### 肘部法

肘部法可以用来确定 K 值.

- 对于n个点的数据集，迭代计算 k from 1 to n，每次聚类完成后计算 SSE 
- SSE 是会逐渐变小的，因为每个点都是它所在的簇中心本身。
- SSE 变化过程中会出现一个拐点，下降率突然变缓时即认为是最佳 n_clusters 值。
- 在决定什么时候停止训练时，肘形判据同样有效，数据通常有更多的噪音，在增加分类无法带来更多回报时，我们停止增加类别。

<img src="D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day09-%E8%81%9A%E7%B1%BBkmeans%E7%AE%97%E6%B3%95+SVM/03-%E7%AC%94%E8%AE%B0/images/image-20230907113314803.png" alt="image-20230907113314803" style="zoom:50%;" />

### 轮廓系数

衡量每个样本与自身所在簇的相似度（簇内紧密性）和与其他簇的相似度（簇间分离度）。

- 取值范围：$[-1, 1]$
  - 接近 1：聚类效果优，
  - 接近 0：聚类模糊。
  - 接近 -1：样本可能被分到错误的簇（聚类效果差）。

### CH 系数(越大越好)

通过簇间离散度与簇内离散度的比值衡量聚类质量，比值越大，聚类越优。

```python

wss_val = kmeans.inertia_  # K-means的目标函数值（簇内总平方和）
```

# 支持向量机SVM

## 1.支持向量机

### SVM的定义

**SVM全称是支持向量机，即寻找到一个超平面使样本分成两类，并且间隔最大。**

<img src="D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day09-%E8%81%9A%E7%B1%BBkmeans%E7%AE%97%E6%B3%95+SVM/03-%E7%AC%94%E8%AE%B0/images/image-20230907155134223.png" alt="image-20230907155134223" style="zoom:50%;" />

### 硬间隔

如果样本线性可分，**在所有样本分类都正确的情况下，寻找最大间隔，这就是硬间隔**

如果出现异常值、或者样本不能线性可分，此时硬间隔无法实现。

### 软间隔和惩罚系数

**允许部分样本，在最大间隔之内，甚至在错误的一边，寻找最大间隔，这就是软间隔**

![image-20230907161700948](D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day09-%E8%81%9A%E7%B1%BBkmeans%E7%AE%97%E6%B3%95+SVM/03-%E7%AC%94%E8%AE%B0/images/image-20230907161700948.png)

通过惩罚系数C来控制这个平衡：C值越小，则间隔越宽，但是间隔违例也会越多。

左边使用了高C值，右边使用了低C值，间隔大了很多，但是位于间隔上的实例也更多。

## 2.SVM算法原理

### 目标函数确定

SVM思想：要去求一组参数（w,b），使其构建的超平面函数能够最优地分离两个集合。样本空间中任意点x到超平面（w,b）的距离可写成： 
$$
r = \frac{\vert \boldsymbol{w}^{\mathrm{T}} \boldsymbol{x} + b \vert}{\lVert \boldsymbol{w} \rVert}
$$
<img src="D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day09-%E8%81%9A%E7%B1%BBkmeans%E7%AE%97%E6%B3%95+SVM/03-%E7%AC%94%E8%AE%B0/images/image-20230907235733281.png" alt="image-20230907235733281" style="zoom:50%;" />

欲找到具有最大间隔的划分超平面，也就是要找到能满足下式中约束的参数w和b，使得间隔γ最大。 
$$
\begin{cases}
\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_i + b \geqslant +1, & y_i = +1; \\
\boldsymbol{w}^{\mathrm{T}} \boldsymbol{x}_i + b \leqslant -1, & y_i = -1.
\end{cases}
$$
距离超平面最近的几个训练样本点使上式等号成立，他们被称为“支持向量”，**两个异类支持向量到超平面的距离之和为：** 
$$
\gamma = \frac{2}{\lVert \boldsymbol{w} \rVert}
$$
要最大化间隔，等价于最小化 $\frac{1}{2}\|\boldsymbol{w}\|^2$，同时要满足约束条件 $y_i(\boldsymbol{w}^T\boldsymbol{x}_i + b) \geq 1, i = 1,2,\cdots,n$。这是一个凸二次规划问题，可以使用拉格朗日乘子法将其转化为对偶问题来求解。

### 核函数的作用

核函数，是将原始输入空间映射到新的特征空间，从而，使得原本线性不可分的样本可能在核空间可分。

<img src="D:/download/000%20%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E9%85%8D%E5%A5%97%E8%AF%BE%E4%BB%B6/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0%20-AI%E7%89%88/004%20%E6%9C%BA%E5%99%A8%E5%AD%A6%E4%B9%A0-V5.0-AI%E7%89%88/day09-%E8%81%9A%E7%B1%BBkmeans%E7%AE%97%E6%B3%95+SVM/03-%E7%AC%94%E8%AE%B0/images/image-20230907175708063.png" alt="image-20230907175708063" style="zoom: 67%;" />

### 核函数的分类

当存在线性不可分的场景时，我们需要使用核函数来提高训练样本的维度、或者将训练样本投向高维

- **线性核**：$K(x_i, x_j) = x_i^T x_j$（适用于线性可分数据）。
- **多项式核**：$K(x_i, x_j) = (x_i^T x_j + c)^d$（处理低维非线性数据）。
- **高斯核（RBF）**：$K(x_i, x_j) = \exp(-\gamma \|x_i - x_j\|^2)$（适用于高维非线性数据，自动映射到无限维空间）。

### 高斯核函数

高斯核函数(RBF)，其中γ 为超参数，γ越大，高斯分布越窄，越容易过拟合。
$$
K(x, y) = e^{-\gamma \lVert x - y \rVert^2}
$$

**sklearn.svm.SVC**：不仅可以处理线性分类问题，还能借助核函数将数据映射到高维空间，进而处理非线性分类问题。当使用线性核（`kernel='linear'` ）时，其原理和`LinearSVC`类似。

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn import datasets
from sklearn.svm import SVC
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import accuracy_score

# 1. 生成非线性可分数据（月牙形数据）
X, y = datasets.make_moons(n_samples=100, noise=0.1, random_state=42)

# 2. 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42
)

# 3. 特征标准化（SVM对特征尺度敏感）
scaler = StandardScaler()
X_train_scaled = scaler.fit_transform(X_train)
X_test_scaled = scaler.transform(X_test)

# 4. 初始化SVM模型（使用RBF核处理非线性数据）
svm_model = SVC(kernel='rbf', C=1.0, gamma='scale', random_state=42)

# 5. 训练模型
svm_model.fit(X_train_scaled, y_train)

# 6. 预测与评估
y_pred = svm_model.predict(X_test_scaled)
accuracy = accuracy_score(y_test, y_pred)
print(f"测试集准确率：{accuracy:.4f}")

# 7. 可视化决策边界
def plot_decision_boundary(model, X, y):
    h = 0.02  # 网格步长
    x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx, yy = np.meshgrid(np.arange(x_min, x_max, h),
                         np.arange(y_min, y_max, h))
    Z = model.predict(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)
    plt.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')
    plt.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='k')
    # 标记支持向量
    plt.scatter(
        X[model.support_, 0], X[model.support_, 1],
        s=100, linewidth=1, facecolors='none', edgecolors='black',
        label='支持向量'
    )
    plt.xlabel('特征1')
    plt.ylabel('特征2')
    plt.title('SVM决策边界（RBF核）')
    plt.legend()
    plt.show()

plot_decision_boundary(svm_model, X_train_scaled, y_train)

```
